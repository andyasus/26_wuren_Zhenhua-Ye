## 一、 研究方向 (Research Direction)

### 1.1 要解决什么问题？
深度学习网络中，**深度 (Depth)** 非常重要，层数越多，能学到的特征层次就越丰富。但实践中发现，当网络深到一定程度（比如超过20层），性能不仅不上升，反而会快速退化 (**Degradation Problem**)。这种退化**不是由过拟合 (Overfitting)** 引起的，因为连训练集的误差都变高了。

### 1.2 为什么会出现这个问题？
直觉上，假如一个20层的网络已经能工作得很好，那么堆出一个56层的网络，只需要让多出来的36层什么也不做，直接把输入原样输出（即**恒等映射，Identity Mapping**）就行了。但现实是，常规的训练方法很难让深层网络学会这种简单的“恒等映射”。

### 1.3 作者的解决思路
作者提出**残差学习 (Residual Learning)** 框架。核心思想是：不让网络直接去学一个复杂的理想函数 \(H(x)\)，而是去学**输入 \(x\) 与理想输出之间的“差异部分”**，即残差函数 \(F(x) = H(x) - x\)。最终输出就是 \(F(x) + x\)。这样，当“什么都不做”是最优解时，网络只需要把 \(F(x)\) 推到零，比学习一个精确的恒等映射容易得多。

## 二、 网络结构 (Network Architecture)

### 2.1 核心积木：残差块 (Residual Block)
网络由一个个**残差块**堆叠而成。一个最基本的残差块结构如下（对应论文 Fig.2）：
```mermaid
    X[输入 x] --> Plus((+))
    X --> Conv1[卷积层1]
    Conv1 --> BN1[批归一化 BN]
    BN1 --> ReLU1[ReLU 激活]
    ReLU1 --> Conv2[卷积层2]
    Conv2 --> BN2[批归一化 BN]
    BN2 --> F[残差 F(x)]
    F --> Plus
    Plus --> ReLU3[ReLU 激活]
    ReLU3 --> Y[输出 y]
```

**数学形式：** \(y = F(x, \{W_i\}) + x\)

*   **捷径连接 (Shortcut Connection):** 把输入 \(x\) 直接传到后面去相加，就像一条跳过中间层的“高速公路”。它不增加额外参数。
*   **残差 \(F(x)\):** 中间这些层学习出的“差异”或“补丁”。
*   **相加与激活:** 两者相加后，再通过一个 ReLU 激活函数。

### 2.2 维度不匹配怎么办？
当输入 \(x\) 和残差 \(F(x)\) 的维度（长宽、通道数）不同时，需要调整捷径上的 \(x\) 的维度：
*   **方案A (零填充):** 直接补零对齐，不增加参数。
*   **方案B (1×1卷积投影):** 用一个 1×1 的卷积层将 \(x\) 变换到与 \(F(x)\) 同一维度，会引入少量参数。论文主要采用此方案处理维度变化。

### 2.3 针对极深网络的“瓶颈”设计 (Bottleneck)
为了构建50层以上的网络同时节省计算量，作者设计了**瓶颈残差块**（论文 Fig.5 右）：
`1×1卷积 (降维) -> 3×3卷积 (特征提取) -> 1×1卷积 (升维恢复)`
先用1×1压缩通道（瓶颈变细），做昂贵的3×3计算，再用1×1恢复。这样计算更高效。

### 2.4 完整的网络
整个ResNet就是由上述残差块堆叠而成（如 ResNet-34, ResNet-50, ResNet-101, ResNet-152），最后接一个**全局平均池化层 (Global Average Pooling)** 和一个1000类的**全连接层 (FC) + Softmax** 进行分类。

## 三、 训练方式 (Training Methodology)

*   **数据增强 (Data Augmentation):**  
    训练时对图像进行随机缩放（短边256-480）、随机裁剪224×224、随机水平翻转、颜色扰动。这能人为扩充数据集，减少过拟合。
*   **优化器与参数:**  
    *   使用带动量 (**Momentum=0.9**) 的随机梯度下降 (**SGD**)。
    *   **Mini-batch size** 为 256。
    *   初始**学习率 (Learning Rate)** 为 0.1，当误差停滞时除以10。
    *   **权重衰减 (Weight Decay)** 设为 0.0001。
    *   不采用 Dropout（因为有BN了）。
*   **归一化与初始化:**  
    *   每个卷积层之后、ReLU 激活之前都加上了**批归一化 (Batch Normalization, BN)**，极大地稳定了训练。
    *   **权重初始化**采用 MSRA 初始化方法。

## 四、 数据集构成 (Dataset Composition)

### 4.1 ImageNet (大规模图像分类)
*   **任务:** 1000类物体分类。
*   **规模:** 训练集约128万张，验证集5万张，测试集10万张。
*   **评价指标:** Top-1 错误率 和 Top-5 错误率。152层 ResNet 的集成模型在 ILSVRC 2015 上取得了 3.57% 的 Top-5 错误率冠军成绩。

### 4.2 CIFAR-10 (小规模分析与验证)
*   **任务:** 10类32x32彩色图片分类。
*   **规模:** 训练集5万张，测试集1万张。
*   **用途:** 在更简单的数据集上验证极深网络（100层甚至1000层）的行为，并进行层响应分析，证明残差网络确实更容易让深层网络学到东西。
