# 作业一：锥桶图像分类

## 说明
训练模型写在 `train.py` 中；**运行 `test.py` 时显示报错没有 `net.py` 文件，因此重新写了 `my_test.py` 文件来验证训练模型。**

## 代码流程
1. 加载数据（`dataset/train`, `test1`, `test2`）
2. 数据增强：随机水平翻转、随机旋转
3. 搭建网络：3个卷积层 + 3个全连接层（共6层参数层）
4. 训练15个epoch，保存最佳模型 `best_model.pth`
5. 在 `test1` 和 `test2` 上验证，打印整体及每类准确率

## 网络结构
- 输入: 3通道, 128×128
- `Conv2d(3, 32, 3) -> ReLU -> MaxPool -> (32, 64, 64)`
- `Conv2d(32, 64, 3) -> ReLU -> MaxPool -> (64, 32, 32)`
- `Conv2d(64, 128, 3) -> ReLU -> MaxPool -> (128, 16, 16)`
- `Flatten -> Linear(32768, 256) -> ReLU`
- `Linear(256, 128) -> ReLU`
- `Linear(128, 3) -> 输出`

## 验证结果
**Test1:**
整体准确率: 88.02%
- blue: 100.00% (样本数: 83)
- red: 70.37% (样本数: 81)
- yellow: 96.23% (样本数: 53)

**Test2:**
整体准确率: 95.00%
- blue: 98.00% (样本数: 50)
- red: 95.92% (样本数: 49)
- yellow: 90.24% (样本数: 41)

## 创新点
- 数据增强：随机水平翻转 + 随机旋转（防止过拟合）
- 全连接层不超过3层，模型轻量化

## 一、深度学习基础知识笔记（`deep_learning_notes.md`）

### 学习总结
- 深度学习每一层都有明确分工：卷积提取特征、池化降维、全连接做分类

- 防止过拟合的手段包括：Dropout、批归一化（BN）、数据增强等
