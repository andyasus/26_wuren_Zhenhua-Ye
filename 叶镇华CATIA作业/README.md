# 叶镇华 CATIA作业 航模发动机建模
## 说明：
- 打开Sys_Model_Engine.CATProduct文件时装配体及各部件应当可以正常显示，**但可能会出现弹窗“未找到以下列表的文件”**；
- 这是因为我装配完成之后发现部分零件文件命名有错误，然后更改了部分零件文件名称。实在非常抱歉，这不符合建模的规范；
- 不过我补救了一下，使用“编辑”-“链接”功能重新加载相应的零件，使用CATDUA修复错误，发现CATIA寻找元素时可以通过零件本身的UUID找到对应路径的文件，因此不影响装配体的正常显示；
- 文件夹中装配体和所有零件使用CAITA的“发送到”功能另存；
- 我在同学的电脑上也验证了一遍我的装配体是可以正常打开的，希望我这份历经十几个小时完成的建模作业，学长/学姐您可以正常打开并检查，谢谢！
